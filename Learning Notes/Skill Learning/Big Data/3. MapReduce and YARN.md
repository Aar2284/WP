## 1. The MapReduce Programming Model

**MapReduce** is a software framework and programming model designed for processing vast amounts of data in parallel on large clusters of commodity hardware in a reliable, fault-tolerant manner. 

### Core Concept: Divide and Conquer
MapReduce breaks down processing into two primary logical phases:
1. **Map**: Filter and sort data. It processes input data blocks and converts them into intermediate key-value pairs.
2. **Reduce**: Aggregate data. It summarizes the intermediate key-value pairs to produce the final output.

### Data Flow Pipeline
The lifecycle of a MapReduce job involves several stages:

```
[Input Data on HDFS]
         |
         v
   [InputFormat] ---- (Splits files into InputSplits)
         |
         v
    [RecordReader] ---- (Parses splits into K1, V1)
         |
         v
     [Mapper] ---- (Processes K1, V1 -> Emits K2, V2)
         |
         v
   [Combiner] ---- (Optional: Local aggregation of K2, V2 on mapper node)
         |
         v
[Shuffle & Sort] ---- (Partitions K2, V2 by Key and transfers to Reducers)
         |
         v
    [Reducer] ---- (Processes K2, List[V2] -> Emits K3, V3)
         |
         v
  [OutputFormat] ---- (Writes K3, V3 to HDFS output directory)
```

1. **Input Splits**: The framework splits the input files into logical chunks called **Input Splits** (typically matching the HDFS block size of 128MB).
2. **RecordReader**: For each Input Split, a RecordReader parses the raw bytes into a stream of key-value records (e.g., line offset as Key, line text as Value).
3. **Map**: The user-defined `map()` function processes these records, emitting intermediate `(Key2, Value2)` records.
4. **Combine (Optional)**: A mini-reducer running locally on each mapper node that performs early aggregation on mapper outputs to reduce network transit volume.
5. **Shuffle & Sort (Network Transfer)**:
   * **Partitioning**: A partitioner (typically `HashPartitioner`) determines which Reducer will process which key: `hash(key) % number_of_reducers`.
   * **Sorting**: The framework sorts all intermediate key-value pairs by key before they reach the Reducer, grouping identical keys together.
6. **Reduce**: The user-defined `reduce()` function receives a key and an iterator of all values mapped to that key `(Key2, List(Value2))`. It computes the final aggregation and emits `(Key3, Value3)`.
7. **OutputFormat**: Writes the final outputs to files in HDFS (typically named `part-r-00000`).

## 2. Java MapReduce API Breakdown

A complete Java MapReduce program consists of three classes: the **Mapper**, the **Reducer**, and the **Driver**.

### A. The Mapper Class
The Mapper class extends `org.apache.hadoop.mapreduce.Mapper` and defines the input/output types as generics:
```java
public class MyMapper extends Mapper<KeyIn, ValueIn, KeyOut, ValueOut> {
    @Override
    protected void map(KeyIn key, ValueIn value, Context context) 
            throws IOException, InterruptedException {
        // Business logic here
        context.write(KeyOut, ValueOut);
    }
}
```
* **Generics Mapping**:
  * `KeyIn`: Input key type (typically `LongWritable` - byte offset of the line).
  * `ValueIn`: Input value type (typically `Text` - the content of the line).
  * `KeyOut`: Output intermediate key type (e.g., `Text` for a word).
  * `ValueOut`: Output intermediate value type (e.g., `IntWritable` for a count of 1).

### B. The Reducer Class
The Reducer class extends `org.apache.hadoop.mapreduce.Reducer`:
```java
public class MyReducer extends Reducer<KeyIn, ValueIn, KeyOut, ValueOut> {
    @Override
    protected void reduce(KeyIn key, Iterable<ValueIn> values, Context context) 
            throws IOException, InterruptedException {
        // Aggregation logic here
        context.write(KeyOut, ValueOut);
    }
}
```
* **Generics Mapping**:
  * `KeyIn` & `ValueIn`: Must match `KeyOut` & `ValueOut` emitted by the Mapper.
  * `KeyOut` & `ValueOut`: Final output types written to HDFS.
  * `Iterable<ValueIn>`: A grouped list of all values associated with the input `key`.

### C. The Driver Class
The Driver configures the MapReduce job configuration and coordinates execution on the cluster.
* Uses the `Job` class to define the configurations.
* Specifies the Mapper, Reducer, and Combiner classes.
* Defines the Map output and final output key-value types.
* Specifies the directory paths for input and output datasets on HDFS.

## 3. YARN: Yet Another Resource Negotiator

In Hadoop 2.x and 3.x, **YARN** acts as the dedicated operating system layer for resource allocation and management. It separates resource allocation from execution tracking, allowing multiple paradigms (MapReduce, Spark, Flink) to co-exist.

### Core YARN Components

```
                =====================================
                |      YARN ResourceManager (RM)     |
                =====================================
                    /                           \
       (Resource Requests)                 (Resource Requests)
                  /                               \
                 v                                 v
   ===========================        ===========================
   |       NodeManager       |        |       NodeManager       |
   |   +-----------------+   |        |   +-----------------+   |
   |   |    Container    |   |        |   |    Container    |   |
   |   | [AppMaster MR]  |   |        |   |   [Map Task]    |   |
   |   +-----------------+   |        |   +-----------------+   |
   ===========================        ===========================
          Physical Node 1                    Physical Node 2
```

1. **ResourceManager (RM)**: The global master coordinator.
   * **Scheduler**: Pure resource distributor based on capacity/queue limits. Does not monitor application state or task failures.
   * **ApplicationsManager (ASM)**: Coordinates incoming job submissions. It negotiates the execution of the first **Container** on a NodeManager to host the application-specific **ApplicationMaster** daemon.
2. **NodeManager (NM)**: The slave daemon running on each machine. It tracks local node hardware usage (RAM, CPU cores) and manages **Containers** as instructed by the ResourceManager.
3. **ApplicationMaster (AM)**: A lightweight process spawned per application/job.
   * It negotiates resource containers from the ResourceManager.
   * It contacts NodeManagers directly to launch application tasks (such as individual Map or Reduce tasks).
   * Monitors execution status, handles faults, and shuts down when the job finishes.
4. **Container**: An isolated physical resource bundle (allocated RAM, CPU vCores, and environment profiles) allocated on a worker node where tasks are executed.

### YARN Job Submission Workflow
1. **Client Submission**: The client program submits a MapReduce job to the YARN ResourceManager.
2. **Launch ApplicationMaster**: The ResourceManager allocates a Container on a NodeManager and boots the job's **ApplicationMaster**.
3. **Resource Estimation**: The ApplicationMaster reads the input splits from HDFS, calculates the number of Map and Reduce tasks required, and requests Containers from the ResourceManager.
4. **Container Launch**: The ApplicationMaster receives Container allocations, contacts the target NodeManagers, and launches the tasks (e.g., executing Java Mapper/Reducer threads).
5. **Task Monitoring**: The tasks stream their execution progress back to the ApplicationMaster.
6. **Cleanup**: Once all tasks complete, the ApplicationMaster releases its container resources back to YARN and reports completion to the ResourceManager.

### Program 1: Classic Word Count
Counts the occurrences of each word in a set of text files.

```java
package bigdata.mapreduce;

import java.io.IOException;
import java.util.StringTokenizer;
import org.apache.hadoop.conf.Configuration;
import org.apache.hadoop.fs.Path;
import org.apache.hadoop.io.IntWritable;
import org.apache.hadoop.io.LongWritable;
import org.apache.hadoop.io.Text;
import org.apache.hadoop.mapreduce.Job;
import org.apache.hadoop.mapreduce.Mapper;
import org.apache.hadoop.mapreduce.Reducer;
import org.apache.hadoop.mapreduce.lib.input.FileInputFormat;
import org.apache.hadoop.mapreduce.lib.output.FileOutputFormat;

public class WordCount {

    // Mapper Class
    public static class WordCountMapper 
            extends Mapper<LongWritable, Text, Text, IntWritable> {
        
        private final static IntWritable one = new IntWritable(1);
        private Text word = new Text();

        @Override
        protected void map(LongWritable key, Text value, Context context) 
                throws IOException, InterruptedException {
            // Split line into tokens/words
            StringTokenizer itr = new StringTokenizer(value.toString());
            while (itr.hasMoreTokens()) {
                // Normalize word: remove punctuation and convert to lowercase
                String cleanedWord = itr.nextToken().replaceAll("[^a-zA-Z]", "").toLowerCase();
                if (!cleanedWord.isEmpty()) {
                    word.set(cleanedWord);
                    context.write(word, one);
                }
            }
        }
    }

    // Reducer Class
    public static class WordCountReducer 
            extends Reducer<Text, IntWritable, Text, IntWritable> {
        
        private IntWritable result = new IntWritable();

        @Override
        protected void reduce(Text key, Iterable<IntWritable> values, Context context) 
                throws IOException, InterruptedException {
            int sum = 0;
            for (IntWritable val : values) {
                sum += val.get();
            }
            result.set(sum);
            context.write(key, result);
        }
    }

    // Driver Code
    public static void main(String[] args) throws Exception {
        if (args.length != 2) {
            System.err.println("Usage: WordCount <input path> <output path>");
            System.exit(-1);
        }

        Configuration conf = new Configuration();
        Job job = Job.getInstance(conf, "Word Count");
        
        job.setJarByClass(WordCount.class);
        job.setMapperClass(WordCountMapper.class);
        job.setCombinerClass(WordCountReducer.class);
        job.setReducerClass(WordCountReducer.class);

        job.setOutputKeyClass(Text.class);
        job.setOutputValueClass(IntWritable.class);

        FileInputFormat.addInputPath(job, new Path(args[0]));
        FileOutputFormat.setOutputPath(job, new Path(args[1]));

        System.exit(job.waitForCompletion(true) ? 0 : 1);
    }
}
```

### Program 2: Sum of Even Numbers
Reads a file containing a list of integers and calculates the sum of only the even numbers.

```java
package bigdata.mapreduce;

import java.io.IOException;
import java.util.StringTokenizer;
import org.apache.hadoop.conf.Configuration;
import org.apache.hadoop.fs.Path;
import org.apache.hadoop.io.IntWritable;
import org.apache.hadoop.io.LongWritable;
import org.apache.hadoop.io.Text;
import org.apache.hadoop.mapreduce.Job;
import org.apache.hadoop.mapreduce.Mapper;
import org.apache.hadoop.mapreduce.Reducer;
import org.apache.hadoop.mapreduce.lib.input.FileInputFormat;
import org.apache.hadoop.mapreduce.lib.output.FileOutputFormat;

public class SumOfEvenNumbers {

    // Mapper Class
    public static class EvenNumberMapper 
            extends Mapper<LongWritable, Text, Text, IntWritable> {
        
        private final static Text evenLabel = new Text("EvenSum");

        @Override
        protected void map(LongWritable key, Text value, Context context) 
                throws IOException, InterruptedException {
            StringTokenizer itr = new StringTokenizer(value.toString());
            while (itr.hasMoreTokens()) {
                try {
                    int num = Integer.parseInt(itr.nextToken().trim());
                    // Check if the number is even
                    if (num % 2 == 0) {
                        context.write(evenLabel, new IntWritable(num));
                    }
                } catch (NumberFormatException e) {
                    // Ignore non-integer elements
                }
            }
        }
    }

    // Reducer Class
    public static class EvenSumReducer 
            extends Reducer<Text, IntWritable, Text, IntWritable> {
        
        private IntWritable finalSum = new IntWritable();

        @Override
        protected void reduce(Text key, Iterable<IntWritable> values, Context context) 
                throws IOException, InterruptedException {
            int sum = 0;
            for (IntWritable val : values) {
                sum += val.get();
            }
            finalSum.set(sum);
            context.write(key, finalSum);
        }
    }

    // Driver Code
    public static void main(String[] args) throws Exception {
        if (args.length != 2) {
            System.err.println("Usage: SumOfEvenNumbers <input path> <output path>");
            System.exit(-1);
        }

        Configuration conf = new Configuration();
        Job job = Job.getInstance(conf, "Sum Of Even Numbers");
        
        job.setJarByClass(SumOfEvenNumbers.class);
        job.setMapperClass(EvenNumberMapper.class);
        job.setReducerClass(EvenSumReducer.class);

        job.setOutputKeyClass(Text.class);
        job.setOutputValueClass(IntWritable.class);

        FileInputFormat.addInputPath(job, new Path(args[0]));
        FileOutputFormat.setOutputPath(job, new Path(args[1]));

        System.exit(job.waitForCompletion(true) ? 0 : 1);
    }
}
```

### Program 3: Palindrome Checker & Counter
Identifies words that are palindromes and counts how many times each palindrome word occurs.

```java
package bigdata.mapreduce;

import java.io.IOException;
import java.util.StringTokenizer;
import org.apache.hadoop.conf.Configuration;
import org.apache.hadoop.fs.Path;
import org.apache.hadoop.io.IntWritable;
import org.apache.hadoop.io.LongWritable;
import org.apache.hadoop.io.Text;
import org.apache.hadoop.mapreduce.Job;
import org.apache.hadoop.mapreduce.Mapper;
import org.apache.hadoop.mapreduce.Reducer;
import org.apache.hadoop.mapreduce.lib.input.FileInputFormat;
import org.apache.hadoop.mapreduce.lib.output.FileOutputFormat;

public class PalindromeCounter {

    // Mapper Class
    public static class PalindromeMapper 
            extends Mapper<LongWritable, Text, Text, IntWritable> {
        
        private final static IntWritable one = new IntWritable(1);
        private Text wordText = new Text();

        // Helper method to check if a word is a palindrome
        private boolean isPalindrome(String str) {
            if (str == null || str.length() < 2) {
                return false; // Skip single characters or empty strings
            }
            int left = 0;
            int right = str.length() - 1;
            while (left < right) {
                if (str.charAt(left) != str.charAt(right)) {
                    return false;
                }
                left++;
                right--;
            }
            return true;
        }

        @Override
        protected void map(LongWritable key, Text value, Context context) 
                throws IOException, InterruptedException {
            StringTokenizer itr = new StringTokenizer(value.toString());
            while (itr.hasMoreTokens()) {
                String rawWord = itr.nextToken().replaceAll("[^a-zA-Z]", "").toLowerCase();
                if (isPalindrome(rawWord)) {
                    wordText.set(rawWord);
                    context.write(wordText, one);
                }
            }
        }
    }

    // Reducer Class
    public static class PalindromeReducer 
            extends Reducer<Text, IntWritable, Text, IntWritable> {
        
        private IntWritable result = new IntWritable();

        @Override
        protected void reduce(Text key, Iterable<IntWritable> values, Context context) 
                throws IOException, InterruptedException {
            int sum = 0;
            for (IntWritable val : values) {
                sum += val.get();
            }
            result.set(sum);
            context.write(key, result);
        }
    }

    // Driver Code
    public static void main(String[] args) throws Exception {
        if (args.length != 2) {
            System.err.println("Usage: PalindromeCounter <input path> <output path>");
            System.exit(-1);
        }

        Configuration conf = new Configuration();
        Job job = Job.getInstance(conf, "Palindrome Counter");
        
        job.setJarByClass(PalindromeCounter.class);
        job.setMapperClass(PalindromeMapper.class);
        job.setCombinerClass(PalindromeReducer.class);
        job.setReducerClass(PalindromeReducer.class);

        job.setOutputKeyClass(Text.class);
        job.setOutputValueClass(IntWritable.class);

        FileInputFormat.addInputPath(job, new Path(args[0]));
        FileOutputFormat.setOutputPath(job, new Path(args[1]));

        System.exit(job.waitForCompletion(true) ? 0 : 1);
    }
}
```

### Program 4: Factorial Calculator
Reads a list of positive integers and outputs the factorial value of each integer.

```java
package bigdata.mapreduce;

import java.io.IOException;
import java.math.BigInteger;
import java.util.StringTokenizer;
import org.apache.hadoop.conf.Configuration;
import org.apache.hadoop.fs.Path;
import org.apache.hadoop.io.LongWritable;
import org.apache.hadoop.io.Text;
import org.apache.hadoop.mapreduce.Job;
import org.apache.hadoop.mapreduce.Mapper;
import org.apache.hadoop.mapreduce.Reducer;
import org.apache.hadoop.mapreduce.lib.input.FileInputFormat;
import org.apache.hadoop.mapreduce.lib.output.FileOutputFormat;

public class FactorialCalculator {

    // Mapper Class
    public static class FactorialMapper 
            extends Mapper<LongWritable, Text, Text, Text> {
        
        // Helper method to compute factorial using BigInteger to handle large outputs
        private String computeFactorial(int n) {
            if (n < 0) return "Undefined";
            BigInteger fact = BigInteger.ONE;
            for (int i = 1; i <= n; i++) {
                fact = fact.multiply(BigInteger.valueOf(i));
            }
            return fact.toString();
        }

        @Override
        protected void map(LongWritable key, Text value, Context context) 
                throws IOException, InterruptedException {
            StringTokenizer itr = new StringTokenizer(value.toString());
            while (itr.hasMoreTokens()) {
                try {
                    int val = Integer.parseInt(itr.nextToken().trim());
                    if (val >= 0) {
                        String factString = computeFactorial(val);
                        context.write(new Text("Num: " + val), new Text(factString));
                    }
                } catch (NumberFormatException e) {
                    // Ignore non-integers
                }
            }
        }
    }

    // Reducer Class
    public static class FactorialReducer 
            extends Reducer<Text, Text, Text, Text> {
        
        @Override
        protected void reduce(Text key, Iterable<Text> values, Context context) 
                throws IOException, InterruptedException {
            // Since factorials are computed directly in Mapper, just write out key and value
            for (Text val : values) {
                context.write(key, val);
            }
        }
    }

    // Driver Code
    public static void main(String[] args) throws Exception {
        if (args.length != 2) {
            System.err.println("Usage: FactorialCalculator <input path> <output path>");
            System.exit(-1);
        }

        Configuration conf = new Configuration();
        Job job = Job.getInstance(conf, "Factorial Calculator");
        
        job.setJarByClass(FactorialCalculator.class);
        job.setMapperClass(FactorialMapper.class);
        job.setReducerClass(FactorialReducer.class);

        job.setOutputKeyClass(Text.class);
        job.setOutputValueClass(Text.class);

        FileInputFormat.addInputPath(job, new Path(args[0]));
        FileOutputFormat.setOutputPath(job, new Path(args[1]));

        System.exit(job.waitForCompletion(true) ? 0 : 1);
    }
}
```

### Program 5: Armstrong Number Checker
Reads a list of integers and checks if each number is an Armstrong number (an $n$-digit number equal to the sum of the $n$-th power of its digits, e.g., $153 = 1^3 + 5^3 + 3^3$).

```java
package bigdata.mapreduce;

import java.io.IOException;
import java.util.StringTokenizer;
import org.apache.hadoop.conf.Configuration;
import org.apache.hadoop.fs.Path;
import org.apache.hadoop.io.LongWritable;
import org.apache.hadoop.io.Text;
import org.apache.hadoop.mapreduce.Job;
import org.apache.hadoop.mapreduce.Mapper;
import org.apache.hadoop.mapreduce.Reducer;
import org.apache.hadoop.mapreduce.lib.input.FileInputFormat;
import org.apache.hadoop.mapreduce.lib.output.FileOutputFormat;

public class ArmstrongNumberChecker {

    // Mapper Class
    public static class ArmstrongMapper 
            extends Mapper<LongWritable, Text, Text, Text> {
        
        // Helper method to check if a number is an Armstrong Number
        private boolean isArmstrong(int number) {
            if (number < 0) return false;
            
            String numStr = String.valueOf(number);
            int numDigits = numStr.length();
            int sum = 0;
            int temp = number;
            
            while (temp > 0) {
                int digit = temp % 10;
                sum += Math.pow(digit, numDigits);
                temp /= 10;
            }
            return sum == number;
        }

        @Override
        protected void map(LongWritable key, Text value, Context context) 
                throws IOException, InterruptedException {
            StringTokenizer itr = new StringTokenizer(value.toString());
            while (itr.hasMoreTokens()) {
                try {
                    int num = Integer.parseInt(itr.nextToken().trim());
                    if (isArmstrong(num)) {
                        context.write(new Text(String.valueOf(num)), new Text("Is Armstrong Number"));
                    } else {
                        context.write(new Text(String.valueOf(num)), new Text("Not Armstrong Number"));
                    }
                } catch (NumberFormatException e) {
                    // Ignore non-integers
                }
            }
        }
    }

    // Reducer Class
    public static class ArmstrongReducer 
            extends Reducer<Text, Text, Text, Text> {
        
        @Override
        protected void reduce(Text key, Iterable<Text> values, Context context) 
                throws IOException, InterruptedException {
            for (Text val : values) {
                context.write(key, val); // Output results
            }
        }
    }

    // Driver Code
    public static void main(String[] args) throws Exception {
        if (args.length != 2) {
            System.err.println("Usage: ArmstrongNumberChecker <input path> <output path>");
            System.exit(-1);
        }

        Configuration conf = new Configuration();
        Job job = Job.getInstance(conf, "Armstrong Number Checker");
        
        job.setJarByClass(ArmstrongNumberChecker.class);
        job.setMapperClass(ArmstrongMapper.class);
        job.setReducerClass(ArmstrongReducer.class);

        job.setOutputKeyClass(Text.class);
        job.setOutputValueClass(Text.class);

        FileInputFormat.addInputPath(job, new Path(args[0]));
        FileOutputFormat.setOutputPath(job, new Path(args[1]));

        System.exit(job.waitForCompletion(true) ? 0 : 1);
    }
}
```
